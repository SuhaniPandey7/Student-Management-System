# 🎓 Student Management System (Java)

A simple console-based Java project to manage student records.

## 🚀 Features
- Add Student
- View Students
- Search Student (coming soon)
- Delete Student (coming soon)

## 🛠️ Tech Used
- Java
- OOP Concepts
- ArrayList

## 📂 Project Structure
- Student.java → Model class
- StudentManager.java → Logic handling
- Main.java → User interface

## ▶️ How to Run
1. Clone the repo
2. Open in VS Code / IntelliJ
3. Run Main.java

## 📌 Future Improvements
- GUI (Java Swing)
- Database integration
